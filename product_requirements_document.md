# 产品需求文档 (PRD) - 利川红茶电商助农系统

## 1. 产品概述
### 1.1 项目背景
本项目旨在为利川红茶打造一个电商助农销售平台，通过互联网技术拓宽销售渠道，帮助当地茶农增加收入，同时弘扬利川红茶文化。

### 1.2 核心价值
- **助农**: 直接连接农户与消费者，展示农户信息，增加产品透明度和故事感。
- **推广**: 宣传利川红茶（冷后浑、玛瑙红等）及其产地文化。
- **便捷**: 提供流畅的在线购买体验。
- **决策支持**: 通过后台仪表盘为管理者提供实时销售数据支持。
- **真实**: 所有商品、轮播图、农户信息均采用真实本地素材，拒绝虚假网络图。

## 2. 用户角色
### 2.1 普通用户 (User)
- 浏览商品，查看产地和农户信息。
- 搜索商品（按名称或产地）。
- 注册、登录。
- 管理购物车（支持多规格独立管理）。
- 下单支付（支持扫码模拟支付，订单状态实时更新）。
- 查看我的订单状态及历史记录。

### 2.2 管理员 (Admin)
- 登录后台管理系统。
- **仪表盘**: 查看今日订单、总销售额、用户数、待发货统计及近7天销量趋势图。
- **商品管理**: 上架、下架、编辑商品信息（含农户信息）。
- **订单管理**: 查看所有订单，编辑订单信息，执行发货操作。
- **用户管理**: 查看、新增、编辑、删除用户。

## 3. 功能需求

### 3.1 认证模块
- **注册**: 用户名、密码、昵称、手机号。
- **登录**: 账号密码登录，返回 Token。

### 3.2 商品模块 (前台)
- **首页轮播**: 展示利川红茶品牌形象（本地高清素材）。
- **商品列表**: 网格展示。
  - 关键信息: 图片、名称、价格。
  - **助农特色**: 必须显著显示“产地”和“农户姓名”。
  - **实时销量**: 显示真实“已售”数量，随订单生成自动增加。
  - **去重逻辑**: 确保首页商品不重复展示。
- **商品搜索**: 支持关键词搜索（名称/产地）。
- **商品详情**: 展示详细描述、库存、添加到购物车、立即购买。
- **图片兜底**: 所有图片加载失败时自动显示默认本地占位图。

### 3.3 购物车 & 订单模块
- **购物车**: 
  - 支持同一商品不同规格（如“标准装”与“礼盒装”）独立添加与管理。
  - 调整数量、移除商品（支持复杂名称商品删除）、计算总价。
  - 数据与后端实时同步，刷新不丢失。
- **结算**: 生成订单。
- **支付模拟**: 
  - 订单生成后可进行模拟支付。
  - 弹出支付二维码（微信/支付宝），点击“我已支付”后自动更新订单状态。
- **我的订单**: 列表展示，状态包括：待支付、待发货、已发货、已完成。支持对“待支付”订单发起二次支付。

### 3.4 后台管理模块
- **仪表盘 (Dashboard)**:
  - **核心指标**: 实时展示今日订单数、总销售额、总用户数、待发货订单数。
  - **数据可视化**: ECharts 折线图展示近7天每日销售额趋势。
  - **快捷入口**: 统计卡片支持点击跳转至对应管理页。
  - **最新动态**: 展示最近10条订单记录。
- **商品管理**:
  - 列表展示（分页、图片预览）。
  - 新增/编辑商品：需录入产地（如毛坝镇）、农户（如王大伯）、分类（冷后浑/玛瑙红）、初始销量。
  - 删除商品。
- **订单管理**:
  - 列表展示所有用户订单，支持状态筛选。
  - **发货**: 对“待发货”订单进行发货操作。
  - **编辑**: 修改订单收货信息、状态及备注。
- **用户管理**:
  - 列表展示系统所有用户。
  - 新增用户（管理员添加）。
  - 编辑/删除用户信息。

## 4. 非功能需求
- **架构**: 前后端分离。
- **性能**: 页面加载流畅，API 响应迅速。
- **易用性**: 界面简洁大气，突出红茶文化元素（红色调）。
- **数据**: 使用 H2 内存数据库方便演示，但需保证数据模型可迁移至 MySQL。
